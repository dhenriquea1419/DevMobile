# 🚀 INÍCIO RÁPIDO - App "Meu Perfil"

## ⚡ Para os apressados (5 minutos)

### 1. Abra o Terminal PowerShell
```powershell
cd "c:\Users\Daniel Araújo\Documents\devmobile\MeuPerfil"
```

### 2. Inicie o App
```powershell
npx expo start --web
```

### 3. Pronto! 🎉
- O app abrirá automaticamente em `http://localhost:19006`
- Você verá seu perfil com:
  - Foto
  - Nome "Daniel Araújo"
  - Cursos (Dev React Native, Dev Java, QA)
  - Frase motivacional
  - Botão interativo

---

## 📱 Para usar em seu celular

### Se tem o Expo Go instalado:
```powershell
npx expo start
# Escaneie o QR code com seu celular
```

### Se quer usar emulador:
```powershell
npx expo start
# Pressione 'a' para Android emulator
# Ou 'i' para iOS simulator (macOS)
```

## 🆘 Problemas Comuns

| Problema | Solução |
|----------|---------|
| Comando não reconhecido | Instale Node.js ou restart terminal |
| App não inicia | Execute `npm install` primeiro |
| Imagem não carrega | Verifique conexão internet |
| Mudanças não aparecem | Pressione 'r' no terminal para recarregar |
