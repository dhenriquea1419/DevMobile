# 🎯 Seu App React Native - "Meu Perfil"

## ✅ O Que Foi Criado

Um aplicativo React Native completo chamado **"Meu Perfil"** que serve como cartão digital de apresentação pessoal.

## 📱 Recursos Principais

### Componentes Utilizados:
- ✅ **View** - Container para organizar elementos
- ✅ **Text** - Exibição de nome, curso e frase motivacional
- ✅ **Image** - Foto de perfil circular
- ✅ **Button** - Botão interativo com alerta
- ✅ **ScrollView** - Suporte para telas pequenas
- ✅ **Alert** - Mensagem de boas-vindas ao clicar

### Estilização com StyleSheet:
- ✅ Cores personalizadas (roxo #6C5CE7)
- ✅ Card com sombra e border-radius
- ✅ Imagem circular de perfil
- ✅ Layout centralizado e responsivo
- ✅ Espaçamento e tipografia bem definidos

## 🗂️ Estrutura do Projeto

```
MeuPerfil/
```

## 🚀 Como Rodar

### Para Iniciar (escolha uma opção):

#### Opção 1: No Expo Go (Smartphone)
```bash
cd c:\Users\Daniel Araújo\Documents\devmobile\MeuPerfil
npx expo start
```
Escaneie o QR code com seu telefone.

#### Opção 2: No Navegador Web
```bash
cd c:\Users\Daniel Araújo\Documents\devmobile\MeuPerfil
npx expo start --web
```
Abre automaticamente em `http://localhost:19006`

#### Opção 3: Android Emulator
```bash
cd c:\Users\Daniel Araújo\Documents\devmobile\MeuPerfil
npx expo start
# Pressione 'a' no terminal
```

#### Opção 4: iOS Simulator (macOS)
```bash
cd c:\Users\Daniel Araújo\Documents\devmobile\MeuPerfil
npx expo start
# Pressione 'i' no terminal
```
